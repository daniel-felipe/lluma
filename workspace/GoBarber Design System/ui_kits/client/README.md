# Client Booking

Mobile web app for the customer. Three-screen booking flow.

## Screens
- **Home** — dark hero with location + search, category chips, top barbers nearby.
- **Booking** — barber portrait header, service selector, time-slot grid, sticky bottom CTA.
- **Confirmed** — green check, booking summary, calendar/done.

## Flow
Home → tap barber card → Booking → pick service + time → tap "Book chair" → Confirmed → Done returns to Home.
